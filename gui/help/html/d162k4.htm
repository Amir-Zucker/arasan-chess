Appearance

The Appearance dialog allows you to set the following options:

  • Piece font A combo box allows you to set the font used to display pieces.
  • Size Four different board sizes are available: Small, Medium, Large, and Extra Large (XLarge). Note: you can also resize the board by grabbing the corner of it with the mouse and stretching it. When you let go of the mouse, the board will resize to the nearest of the available sizes.
  • Show coordinates: If checked, shows the coordinates (letters and numbers) on the sides of the board.
