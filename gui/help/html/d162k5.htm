Board Colors

The Board Colors menu item brings up a dialog that allows you to set the colors of the squares on the board. This menu item will be enabled only if you are running Arasan with a display that can show colors.

If using a 256-color display, you may get a display somewhat different than the colors you selected, because of the limitations of Windows palettes.

If you check the "Black & white display" checkbox, Arasan will display the board using only black and white (with hatches to show the dark squares), as if running on a mono display.
